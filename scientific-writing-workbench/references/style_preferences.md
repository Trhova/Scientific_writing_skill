# Style Preferences

Use this reference for global, user-specific writing preferences that should apply across projects unless the user says otherwise.

This file is a style layer, not an evidence layer. It should shape phrasing, transitions, figure references, and rhetorical habits, but it must not override scientific accuracy, citation integrity, access-state rules, or venue requirements.

## Precedence

- Direct user instructions in the current chat override this file for that turn.
- Venue or journal requirements override this file when they conflict.
- This file overrides only stylistic defaults, not scientific-integrity rules.

## Figure References

- Prefer making the scientific point first, then attaching the figure reference in parentheses.
- Prefer `(Fig. 1D)` rather than leading with phrases such as `Figure 1D shows...`.
- Use figure references as support for a statement, not as the grammatical subject of the sentence, unless the user explicitly asks for that style.
- Keep figure references unobtrusive. They should help orientation without taking over the sentence.

Preferred:

- `Several microbial metabolites have been linked to both tumor-promoting and tumor-inhibiting effects (Fig. 1D).`
- `These patterns suggest a context-dependent immune effect (Fig. 3B).`

Avoid by default:

- `Figure 1D shows that several microbial metabolites have been linked...`
- `As can be seen in Figure 1D...`

## Tone

- Default to restrained, evidence-aware scientific prose.
- Prefer mechanistic clarity over hype or inflated novelty language.
- Avoid overstating causality, certainty, or generality.
- Prefer precise verbs such as `associated with`, `linked to`, `supports`, `suggests`, `is consistent with`, or `has been reported to` when the evidence is not definitive.

Preferred:

- `These findings are consistent with a metabolite-mediated immune effect.`
- `The microbiome may act as one component of host context rather than a universal determinant of response.`

Avoid by default:

- `This clearly proves...`
- `This dramatically changes...`
- `The microbiome definitively determines...`

## Transitions

- Keep transitions explicit, but do not over-signpost.
- Prefer transitions that explain why the next paragraph follows from the previous one.
- Avoid repetitive meta-phrases such as `Figure X points in this direction`, `As mentioned above`, or `It is important to note that` unless they add real value.

Preferred:

- `This shifts the question from composition to mechanism.`
- `That limitation makes interventional evidence especially important.`
- `This provides a bridge to the next section, where...`

Avoid by default:

- `Figure 1D already points in this direction by...`
- `As stated above...`
- `It should be noted that...`

## Narrative Style

- State the scientific point first, then attach citation or figure support.
- Favor prose that narrows from field-level context to the specific mechanistic question.
- Keep the narrative disciplined: do not over-explain obvious transitions, but do make the argumentative structure visible.
- When multiple interpretations exist, acknowledge them rather than forcing a single clean story.

Preferred:

- `Many microbial metabolites have been linked to cancer-promoting and cancer-inhibiting effects (Fig. 1D).`
- `This makes metabolite-level reasoning more useful than a simple good-microbe versus bad-microbe framing.`

Avoid by default:

- `Figure 1D tells us that...`
- `The point here is that...`
- `This proves the microbiome is important.`

## Allowed And Avoided Patterns

Allowed by default:

- point first, support second
- explicit but compact transitions
- cautious causal language unless strong evidence exists
- figures cited parenthetically

Avoid by default:

- figure-led sentence openings
- hype or grandiose framing
- vague intensifiers
- treating associations as established mechanism
