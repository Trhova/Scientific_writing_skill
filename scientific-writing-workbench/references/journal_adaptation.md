# Journal Adaptation

Use this reference when tailoring a draft to a target journal or conference.

## Inputs to collect

- target venue name
- article type
- word or character limits
- abstract format rules
- reference style
- figure and table caps
- declaration requirements
- supplement policy
- any field-specific reporting checklist

## Adaptation steps

1. Compare the current manuscript structure with the venue's required section order.
2. Check whether the abstract must be structured.
3. Identify missing declarations or compliance statements.
4. Estimate compression pressure from the word limit.
5. Check figure and table counts against venue limits.
6. Convert citation and bibliography style only after the references themselves are validated.

## Extensible venue notes

Keep a simple venue note block:

- venue
- article type
- scope fit notes
- section order
- abstract rules
- citation style
- figure and table rules
- declaration requirements
- open issues

## When the venue instructions are incomplete

- Do not guess hidden requirements.
- Label unknown items as "needs confirmation".
- Adapt only the rules you can verify from supplied instructions or reliable sources.

## Common adaptation outputs

- revised outline matching the venue order
- shortened or expanded abstract
- declaration checklist
- table of sections at risk of exceeding word limits
- submission-readiness note
